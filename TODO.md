
# 階乗
- 1から10までのいずれかの整数を入力すると、その整数の階乗の計算結果が表示される。
    - [x] 1を入力すると1が出力される
    - [x] 2を入力すると2が出力される
    - [x] 3を入力すると6が出力される
    - [x] 4を入力すると24が出力される
    - [x] 10を入力すると3628800が出力される

- それ以外の整数を入力すると-1と出力する。
    - [x] 0を入力すると-1が出力される
    - [x] 11を入力すると-1が出力される


# 閏年
閏年計算ツール

- 西暦年が4で割り切れる年は閏年
    - 2004年は閏年
    - 2003年は平年
- ただし、西暦年が100で割り切れる年は平年
    - 2100年は平年
- ただし、西暦年が400で割り切れる年は閏年
    - 2000年は閏年
